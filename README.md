# Fast IPTV Channel Viewer for Vercel

This version removes `vercel.json` completely.

Vercel does not need a `vercel.json` file for this project because:

- `index.html` is served as the static frontend
- `api/xtream.js` is automatically detected as a serverless function

Upload/deploy this folder as-is.

Files:

- `index.html`
- `api/xtream.js`
- `package.json`
- `README.md`
